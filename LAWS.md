Crie uma aplicação SaaS moderna para uma clínica odontológica chamada OdontoX.
A aplicação deve ser construída seguindo princípios de software moderno, com boa organização de pastas, integração entre frontend e backend e arquitetura limpa.

Tecnologias desejadas

Frontend: React ou Next.js (preferencialmente Next 14 / App Router)

Backend: Node.js (NestJS ou Express)

Banco: PostgreSQL

ORM: Prisma

Autenticação: JWT + Refresh Token

UI/UX: TailwindCSS + shadcn/ui

Padrões: Clean Architecture, SOLID

Módulos que o sistema deve possuir
1. Tela de Login

Login e cadastro

Recuperação de senha

Controle de acesso (admin, doutor, recepção)

2. Dashboard de Atendimento

Cards com: atendimentos do dia, pacientes agendados, consultas em andamento

Agenda estilo Google Calendar

Gráficos de atendimentos semanais/mensais

Notificações internas

3. Cadastro de Doutores

Nome, CRO, especialidade, horário de atendimento

Upload de foto

Status ativo/inativo

Relação com agenda

4. Cadastro de Pacientes

Nome, CPF, telefone, endereço

Histórico odontológico

Anexos (RX, fotos, PDF)

Vínculo com consultas

Sistema de etiquetas (prioridade, retorno, faltou, etc.)

Entregáveis que quero que você gere

Estrutura completa do projeto

Código do backend (rotas, controllers, services, models)

Código do frontend (páginas, componentes, serviços de API)

Schemas Prisma

Design system com shadcn/ui

Documentação em Markdown

Scripts de setup (npm, migrations, seeds)
